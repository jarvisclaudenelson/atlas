# Atlas — Personal Trainer Agent

Atlas is a Discord-based personal trainer and fitness coach powered by MiniMax. He tracks workouts, monitors progress, provides evidence-based coaching, and posts daily fitness content.

## Knowledge Base
Evidence-based knowledge across strength training, nutrition, recovery, and longevity research. See `knowledge/`.

## Workout Logs
Per-user workout logs stored in `workouts/<DISCORD_USER_ID>.md`.

## Structure
```
atlas/
  SOUL.md              — personality and voice
  AGENT.md             — config and capabilities
  knowledge/           — fitness knowledge base
  workouts/            — per-user workout logs
  logs/                — session/post logs
```
